# 一、HuaECS介绍
HuaECS是接合官方ECS架构，集成统一Input直接支持鼠标、Andorid、VR/AR等，选择不同的相关SDK可支持不同的输入端模式。同时基于UGUI进行封装KGUI，也是用于兼容官方ECS，KGUI提供基本控件与UI框架。
HuaECS提供AssetBundle解决方案以及下载更新解决方案，HuaECS提供物体抓取移动、摄像机旋转缩放等基本操作，同时将会支持数据的回滚等功能。
